Функции

void vec_CopyOddToEven_32f (nmreg ar0, nmreg ar6, nmreg gr5)
 

Подробное описание

Ядро функций nmppsCopyOddToEven_32f и nmppsCopyEvenToOdd_32f. Функция копирует числа с плавающей точкой одинарной точности, лежащие с нечетного адреса, на четный:

Аргументы
ar0указатель (может быть только нечетным) на входной массив чисел с плавающей точкой одинарной точности
ar6указатель (может быть только четным) на выходной массив чисел с плавающей точкой одинарной точности
\param gr5

размер (может быть любой кроме 0) входного массива size = [1, 2 ... 31, 32, 33 ...]

Restrictions:
При выходе из функции изменяется содержимое регистров: ar0, ar6.
