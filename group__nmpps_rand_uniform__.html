Подробное описание

Генерация случайного числа с равномерным раcпределением.

Аргументы
nMinМинимальное возможное значение случайного числа.
nMaxМаксимальное возможное значение случайного числа.
nDivisibleЗначение, которому будет кратно случайное число.
Возвращает
int Случайное число в диапазоне либо [nMin, nMax]. Для функции без параметров данный диапазон [-2^31; 2^31-1].
