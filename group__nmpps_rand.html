Подробное описание

Генерация случайного числа с равномерным раcпределением.

Аргументы
nMin
Минимальное возможное значение случайного числа.
nMax
Максимальное возможное значение случайного числа.
nDivisible
Значение, которому будет кратно случайное число.
Возвращает
int Случайное число в диапазоне либо [nMin, nMax]. Для функции без параметров данный диапазон [-2^31; 2^31-1].
